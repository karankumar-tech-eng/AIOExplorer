# AIOExplorer

## Multi-Chain Ethereum Blockchain Explorer

AIOExplorer is a multi-chain blockchain explorer built to track wallet balances, transaction history, token balances, NFTs, and real-time blockchain data across multiple EVM-compatible networks.

## Installation

```bash
git clone https://github.com/karankumar-tech-eng/AIOExplorer.git
cd AIOExplorer
npm install
npm run dev
```

---

## Project Objective

The main objective of this project is to provide a fast, simple, and user-friendly multi-chain blockchain explorer that allows users to monitor blockchain activities across different Ethereum-compatible networks from a single platform.
